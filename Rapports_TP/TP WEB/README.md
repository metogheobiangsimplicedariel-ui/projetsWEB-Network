# LearnSite - Plateforme d'Ateliers Informatiques

## Description

Site web présentant des ateliers de formation en informatique destinés aux étudiants. Développé avec HTML5 et CSS3, le site offre une expérience utilisateur moderne, responsive et accessible.

## Choix de Design

### Palette de Couleurs

#### Couleurs Principales

- **Bleu foncé** (`#242582`) :
  - Représente la **confiance** et le **professionnalisme** du domaine informatique
  - Utilisé pour le header, footer et en-têtes de tableau

- **Rose claire** (`#f64c71`) :
  - Apporte une **énergie moderne** et une touche de dynamisme
  - Attire l'attention sur les calls-to-action (boutons, liens importants) et crée un contraste fort avec le bleu pour une hiérarchie visuelle claire

- **Orange** (`#f67c4c`) :
  - État **hover** des éléments interactifs

#### Couleurs de Support

- **Gris clair** (`#f8f9fa`) : Fond neutre pour ne pas concurrencer le contenu
- **Noir** (`#1a1a1a`) : Texte principal avec contraste optimal pour la lisibilité
- **Gris moyen** (`#6c757d`) : Texte secondaire, labels et informations moins importantes

#### Couleurs Fonctionnelles

- **Vert** (`#28a745`) : États de succès et validation
- **Rouge** (`#dc3545`) : États d'erreur et alertes
- **Blanc** (`#ffffff`) : Surfaces, cartes et arrière-plans de contenu

### Typographie

- **Police principale** : Poppins (Google Fonts)
- **Justification** : Moderne, excellente lisibilité, supports pour les variantes en gras.
- **Polices fallback** : Arial, Helvetica, sans-serif
